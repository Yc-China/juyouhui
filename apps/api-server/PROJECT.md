# 聚优惠 - 餐饮积分联盟平台

## 项目简介

聚优惠是一个面向餐饮行业的通用积分联盟平台：
- 所有商家免费加盟，仅需缴纳 500 元可退保证金
- 商家先充值，1 分钱 = 1 积分，给用户发积分从商家余额扣除
- 用户攒积分到平台兑换礼品（平台自营）
- 积分有效期一年

**优势：**
- 彻底解决传统积分联盟"套礼品"的风险
- 平台现金流健康，先收钱后发货
- 商家零风险，用户有实惠，平台挣差价

## 技术架构

- **服务端：** Node.js + Koa2 + MySQL
- **商家后台：** uniapp 开发微信小程序
- **用户端：** 后续开发，先做商家端

## 核心功能清单

### 商家端
1. **入驻申请**：提交资料，缴纳保证金
2. **门店管理**：多门店管理
3. **充值中心**：给积分账户充值
4. **积分发放**：扫码/输手机号给用户发积分
5. **积分核销**：用户兑换商家商品时核销积分
6. **流水查询**：所有积分变动记录
7. **数据看板**：今日/本月发积分统计

### 平台端（后台管理）
1. **商家审核**：审核入驻申请
2. **保证金管理**：保证金收取/退还
3. **订单管理**：充值订单/兑换订单
4. **礼品管理**：上架/下架兑换礼品
5. **数据统计**：平台整体数据
