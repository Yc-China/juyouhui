
# 聚优惠商家管理后台 (merchant-web)

聚优惠商家管理后台，供入驻商家进行积分管理、用户发积分、充值等操作。

## 技术栈

- React 19 + TypeScript + Vite
- Tailwind CSS
- React Router DOM
- Lucide React (图标库)

## 开发启动

```bash
# 进入项目目录
cd apps/merchant-web

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

开发服务器默认运行在 `http://localhost:3001`

## 构建

```bash
npm run build
```

构建产物输出到 `dist` 目录

## 功能模块

- **登录** - 商家手机号+密码登录
- **首页** - 数据统计卡片 + 快捷入口
- **发积分** - 根据消费金额给用户发积分
- **积分流水** - 查看发积分历史记录
- **充值** - 创建充值订单，查看充值记录
- **我的余额** - 查看积分余额和保证金
- **商家设置** - 设置积分比例、修改密码

## 移动端适配

该项目专为商家移动端使用设计，界面简洁，操作方便，支持手机浏览器直接访问。

## API 代理

开发环境下所有 `/api` 请求会被代理到 `http://localhost:3000` (后端服务地址)
